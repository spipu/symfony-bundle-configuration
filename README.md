# Spipu Configuration Bundle Documentation

## Documentation

This symfony bundle add the following features:

* Configuration Definition
* Configuration Administration

## Requirements

This symfony bundle works with:

* PHP >7.4
* Composer
* Symfony > 5.4
* Spipu Ui Bundle
